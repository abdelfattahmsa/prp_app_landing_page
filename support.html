Frequently Asked Questions

Is PRP really free?

Yes — the Free tier is free forever and includes all 4 core modules (Time, Finance, Energy, Health). A Pro tier with advanced features is coming soon.

Is my data private and secure?

Your data is stored in Supabase with row-level security (RLS). Every row is scoped to your authenticated user — other users cannot see or access your data under any circumstances.

Can I use PRP on multiple devices?

Yes. PRP syncs in real-time via Supabase. Log in on the web app at web.prp-app.website or the Windows desktop app — your data is always in sync.

Will there be a mobile app?

Android, iOS, and macOS apps are all on the roadmap. PRP is built with Flutter so cross-platform support is a natural next step. Follow the roadmap for updates.

How do live stock prices and FX rates work?

PRP uses the Alpha Vantage API for stock prices and a currency exchange API for FX rates. Free-tier Alpha Vantage allows up to 25 requests per day. You can configure your own API key in Profile → Settings → Integrations.

I found a bug — what should I do?

Please open a GitHub issue with a description of the bug, the steps to reproduce it, and your platform (web or Windows). We aim to triage all bugs within 48 hours.

Can I delete my account and data?

Yes. Go to Profile → Settings → Account → Delete Account. This permanently deletes all your data from our Supabase database. This action is irreversible.
